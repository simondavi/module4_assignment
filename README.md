# Course Assignment D1 Modern Tools & Workflows for Data Quality

## Project Structure

The naming scheme of the project folders is as follows:

```
__00_data_raw/         # Sensitive raw data (not shared)
__01_data_processed/   # Anonymized data used in this project
__02_scripts/          # Analysis code (Quarto document)
__03_output/           # Generated outputs (plots)
__04_documentation/    # Scale documentation
```

## Additional Files

- `LICENSE.md` — License information for this project  
- `README.md` — Project overview and documentation  
- `run.sh` — Shell script to open the Quarto html document in the browser
